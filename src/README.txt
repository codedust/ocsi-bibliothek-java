Readme zu den Quellen der OSCI-Bibliothek in der Version 2.4.1

In diesem Verzeichnis sind die Quellen der OSCI-Bibliothek. Sie
lassen sich mit Java-Versionen ab 1.8 übersetzen. Unterstützt 
und getestet wird Java 1.8 und Java 11, es sind aber zum Zeitpunkt
der Auslieferung keine Probleme beim Betrieb mit höheren Java-Versionen
bekannt. Weitere benötigte Software entnehmen Sie bitte dem Ordner mit der 
fertig übersetzten Bibliothek (lib).
